# 151 Trading Strategies (151 Estrategias de Trading)

**Authors:** Zura Kakushadze, Juan Andrés Serur
**Published:** 2019-11-14
**Categories:** q-fin.GN, q-fin.MF, q-fin.PM
**URL:** http://arxiv.org/abs/1912.04492v1
**PDF:** https://arxiv.org/pdf/1912.04492v1

## Abstract
Comprehensive book covering 150+ trading strategies across ALL asset classes and trading styles. 550+ mathematical formulas. Includes stocks, options, fixed income, futures, ETFs, forex, crypto, commodities, volatility trading, etc. Some strategies use ML (ANN, Bayes, KNN). Includes backtesting source code and ~2,000 references. NOTE: Book is in Spanish but formulas are universal.
